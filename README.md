# Siperlik
Siperlik yapımında kullanılan lazer kesim ve 3D baskı dosyaları.



![Siperlik](siperlik.jpg)






## Malzemeler

* 0.5 mm PetG Plaka
* ABS veya PLA flament
* Yatay Lastik
* Zımba


## Baskı Parametreleri
* 1.2 mm Duvar Kalınlığı
* %20 İç Doluluk
* 0.2 mm Katman Kalınlığı
